Clicking will open subscription to three different kinds of observables.

First a cold observable which will start emitting values only after it gets an active subscriber.

Secondly a lukewarm observable, which will run as long as it has active subscribers and in this case new subscribers will tap into the running stream. But if there aren't any more subscribers, it will start from the beginning. (Tip: Click a second time with running subscription)

Thirdly a hot observable. It will always have an active subscriber and new subscribers will always tap into the running stream.
